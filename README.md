# netease-helper
:candy: 网易云音乐小工具。 自动收藏每日日推， 并播放。 支持自动刷个人 FM， 指定歌单。
  
![alt](https://imgcdn.a632079.me/uploads/2018/05/netease_helper.png?ts=1)
  
## 使用
```
# npm i -g pnpm
pnpm i --production
```
然后只需简单得配置一下 `config.json`, 即可正常使用。
